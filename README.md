# OrangeClicker

Very basic C# skeleton for an auto clicker + key presser with a plain WinForms GUI.

## Included imports/libraries

- System
- System.Windows.Forms
- System.Runtime.InteropServices

No external NuGet packages are required for this skeleton.

## Files

- OrangeClicker.csproj
- Program.cs
- MainForm.cs
- Services/InputAutomationService.cs

## Run

1. Install .NET 8 SDK.
2. Run dotnet restore.
3. Run dotnet run.

The click and key methods are placeholders so you can implement your own behavior.
